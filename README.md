# Smart Green Nudging — Causal ML Replication 

## Paper

**Smart Green Nudging: Reducing Product Returns Through Digital Footprints and Causal Machine Learning**  
*Marketing Science, 2022*


## Project

The objective of this project is to replicate the analysis of **treatment heterogeneity** presented in *Smart Green Nudging: Reducing Product Returns Through Digital Footprints and Causal Machine Learning*.

Specifically, this project focuses on re-implementing the paper’s **modern causal machine learning (CML) methods** to estimate heterogeneous treatment effects of green nudges based on consumers’ digital footprint and shopping cart data.
